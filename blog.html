Difference in scope between var, let and const

var has two types of scope

  • 1. functonal
  • 2. global

let has three types of scope

  • 1. block
  • 2. functonal
  • 3. global

const has three types of scope

  • 1. block
  • 2. functonal
  • 3. global

REST API

A REST API (Representational State Transfer Application Programming Interface) is a set of rules and conventions that allows different software applications to communicate with each other over the internet. It defines a standardized way for requesting and exchanging data between systems. In simpler terms, it is a well-defined language that computers use to talk to each other and share information. With REST APIs, one can request specific pieces of data or perform actions on a remote server, making it a fundamental technology for building web and mobile applications that interact with various online services and databases.
